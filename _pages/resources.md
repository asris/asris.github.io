## Pipeline for Integrated Orientation and Neuromapping with Enhanced Exploration and Registration (PIONEER)

<p align="center">
<img src="/assets/img/pioneer_pipeline_grayscale.png" width="900">
</p>

PIONEER is a semi-automated workflow developed to streamline the analysis of mouse brain histological sections. It integrates multiple open-source image processing tools into a single guided pipeline that reduces manual effort while maintaining flexibility and reproducibility.

The platform is designed for researchers performing quantitative neuroanatomical analysis, including atlas registration, cell detection, regional quantification, and visualization from coronal/sagittal brain sections.

---

## Workflow

The analysis pipeline consists of the following steps:

1. **Image Acquisition**
   - Import brightfield or fluorescence images (TIFF, PNG, JPEG).
   - Supports serial mouse coronal/sagittal brain sections.

2. **Atlas Registration**
   - Align histological sections to a reference mouse brain atlas.
   - Semi-automated orientation and anatomical matching.

3. **Image Refinement**
   - Correct tissue distortions and improve atlas correspondence.
   - Allows manual refinement where necessary.

4. **Object Segmentation**
   - Detect labeled cells, nuclei, fibers, or fluorescent signals.
   - Supports both machine-learning and threshold-based segmentation.

5. **Regional Quantification**
   - Quantify detected objects within atlas-defined brain regions.
   - Generate regional counts, densities, and summary statistics.

6. **Visualization & Export**
   - Produce publication-quality overlays.
   - Export annotated images, tables, and quantitative reports.

## Features

- Unified workflow for brain histology analysis
- Semi-automated atlas registration
- Interactive image refinement
- Automated object detection and segmentation
- Region-wise quantitative analysis
- Batch processing support
- Voice-assisted workflow guidance
- Modular architecture
- Offline operation after installation
- Publication-ready outputs
- Reproducible and transparent analysis pipeline

## Applications

PIONEER can be used for a wide range of neuroscience studies, including

- Developmental neuroscience
- Epilepsy research
- Circuit mapping
- Viral tracing experiments
- Immediate early gene (c-Fos) analysis
- Immunohistochemistry
- Cell transplantation studies
- Whole-brain projection analysis
- Mouse disease models
- Quantitative neuroanatomy
